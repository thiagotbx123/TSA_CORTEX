## Decisões Tomadas

|---------|----------|---------------------------|
| TypeScript + Node.js | Melhor tipagem e ecossistema para APIs | Python, Go |
| Commander.js para CLI | Popular e bem documentado | Yargs, Oclif |
| Zod para validação | Tipos runtime + TypeScript | Joi, Yup |
| date-fns-tz | Timezone handling robusto | Luxon, Moment-timezone |
